## Cube of a Number

### Problem
Write a program to calculate cube of a number. Take the number from the user.

### Input / Output
- **Input:** Floating value `num`.
- **Output:** The cube of the number is `cube`

### Approach
- Use: `cube = n * n * n`

### Pseudocode
1. Read `num`
2. cube ← n * n * n
3. Print cube

- **Visual Studio Code** 
  ```bash
  >> gcc cube_of_number.c
  
  >> ./a.exe
